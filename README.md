# SeedboxSync front end

## For developer

### Python part

```bash
python3 -m venv .venv
. .venv/bin/activate
lask --app seedboxsync-front run --debug
```

### Frontweb part

Use [Bulma](https://bulma.io/) / [Vite](https://vite.dev) / [Vanilla](https://vanilla-js.com/):

```bash
npm i
npx run build
```
